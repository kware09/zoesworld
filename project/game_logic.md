# Zoe's World -- Game Logic

## Core Session Flow

Open App ↓ Enter Zoe's World ↓ Play 3 games ↓ Earn stars ↓ Decorate
world ↓ Session complete

Session duration target: 5--8 minutes

------------------------------------------------------------------------

## Star System

1 game completed = 1 star

Rewards: 3 stars → decoration 10 stars → unlock animal

------------------------------------------------------------------------

## Session Rewards

Track sessions played.

Rule:

5 sessions completed → Parent reward unlocked

------------------------------------------------------------------------

## Word Mastery Tracking

Word states:

New Learning Practiced Mastered

Progress rules:

Correct 3 times → Practiced Correct 5 times → Mastered

Mastered words appear less frequently.

------------------------------------------------------------------------

## Difficulty Adjustment

Track:

attempts correct answers response speed

Rules:

If success rate \> 80% over last 10 attempts → increase difficulty

If success rate \< 50% → decrease difficulty

------------------------------------------------------------------------

## Daily Limits

Recommended session structure:

3 games per session Maximum stars per day optional (prevents reward
inflation)

------------------------------------------------------------------------

## Game Rotation

Rotate between:

Robot Talk Sound Ninja Word Explorer

Ensure varied gameplay.
