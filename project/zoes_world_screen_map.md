# Zoe's World -- Application Screen Map

This document outlines the complete navigation structure and key screens
for the Zoe's World application.

The goal is to keep navigation simple and predictable for a 6‑year‑old
user.

------------------------------------------------------------------------

# Global Navigation Principles

The app should always follow a **simple loop**:

Enter World ↓ Play Game ↓ Earn Reward ↓ Return Home

The user should **never get lost** in deep navigation.

Maximum navigation depth: **2 levels**.

------------------------------------------------------------------------

# 1. Home Screen

Purpose: Primary landing screen and central hub.

Components:

• Zoe's world scene (treehouse or village)\
• Current star count\
• Progress indicator toward next reward\
• Buttons:

-   Play a Game
-   Decorate My World
-   My Rewards

Optional visual elements:

• animated animals\
• growing treehouse

------------------------------------------------------------------------

# 2. Game Selection Screen

Purpose: Allow Zoe to choose which mini‑game to play.

Games available:

Robot Talk -- phoneme blending\
Sound Ninja -- sound deletion\
Word Explorer -- decoding practice

Each tile includes:

• icon\
• title\
• optional voice prompt

------------------------------------------------------------------------

# 3. Game Screens

Each game should follow the same layout structure.

Top Area: Character guide (Robot / Ninja / Explorer)

Middle Area: Game prompt (sounds or word)

Bottom Area: Answer choices or interaction controls

Controls:

• Replay sound • Submit answer • Next round

------------------------------------------------------------------------

# 4. Game Result Screen

Purpose: Provide feedback and reward.

Displays:

⭐ Star earned\
Encouraging message\
Progress toward next reward

Buttons:

• Play another game • Return home

------------------------------------------------------------------------

# 5. Decorate My World

Purpose: Allow Zoe to spend stars and personalize her world.

Subsections:

Treehouse

-   Add furniture
-   Upgrade levels

Garden / Meadow

-   Plant flowers
-   Add butterflies

Animals

-   Unlock fox
-   Unlock owl
-   Unlock rabbit

Decorations should be unlocked with stars.

------------------------------------------------------------------------

# 6. My Rewards

Displays:

⭐ Stars collected\
⭐ Progress toward parent reward

Example display:

⭐⭐⭐☆☆

Message example:

"2 more sessions until your reward!"

------------------------------------------------------------------------

# 7. Parent Panel

Hidden entry method:

Tap screen corner **5 times**.

Screens include:

Progress Dashboard

Sessions this week\
Stars earned\
Words mastered

Difficulty Settings

Level 1 -- simple CVC words\
Level 2 -- digraphs\
Level 3 -- blends

Word Manager

Parent can add custom words.

Reward Settings

Configure reward frequency.

------------------------------------------------------------------------

# Screen Flow Example

Typical session:

Home ↓ Play Game ↓ Robot Talk ↓ Result Screen ↓ Play Another Game ↓
Result Screen ↓ Return Home ↓ Decorate World

Session duration target: **5--8 minutes**.

------------------------------------------------------------------------

# UX Design Rules

Large touch targets\
Minimal text\
Voice prompts available\
Gentle animations only

Navigation should always allow:

Return Home
